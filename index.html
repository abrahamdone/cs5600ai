Evolution of 2D cellular automata with a hexagonal rule set

View on GitHub Download .zip Download .tar.gz

Hexagonal grids, 1D cellular automata's, and its two dimensional implementation The Game of Life, come together in my own blend to make a hexagonal game of life. The space is very large and therefore will be immensely hard to find an "interesting" result. I have decided to use a genetic algorithm to search the space to try and find the interesting examples. Evolving the game of life.
