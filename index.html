Evolution

Since the space I have provided is fairly large (2 to the 46th power) it is hard to find good rule sets with a life/death table. I have decided to search for it using evolution. There is a simple genetic algorithm used to evolve a first randomly selected population to try and find a suitable rule.

Set Complexity

I have used the method of set complexity analysis discussed in Kolmogorov complexity of epithelial pattern formation: The role of regulatory network configuration by Nicholas Flann et al. The set complexity is calculated to determine the fitness of each genome. After an initial "burn in" period, the next few states are saved and the set complexity is determined from the complexity.
