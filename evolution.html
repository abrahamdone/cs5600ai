The goal of finding an interesting rule set in this vast space (2 to the 46th power or 70,368,744,177,664 possible rules) is quite daunting. Random search can find some results, but can it actually be searched to find the interesting sets? The solution to this search problem that I thought would be interesting was an evolutionary approach. Effectively evolving a game of life. I utilized a simple genetic algorithm which produced a set of random rule sets as genomes and used this random set evolve an interesting rule set.

Set Complexity

I have used the method of set complexity analysis discussed in Kolmogorov complexity of epithelial pattern formation: The role of regulatory network configuration by Nicholas Flann et al. The set complexity is calculated to determine the fitness of each genome. After an initial "burn in" period, the next few states are saved and the set complexity is determined from the complexity.

The saved states are all compared against each other, by a process of combining and compressing the combined states. The math done on these results in a fitness evaluation. This value is used to determine the overall chance a genome has over another to be selected to pass on to the next generation and combine its rule set with another fit function to produce child rules.

The hope is that this would be a suitable method to find interesting rule sets.
